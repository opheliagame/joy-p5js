# joy.js

this library provides [joy](https://github.com/fossunited/joy), a tiny creative coding library, with [p5.js](http://p5js.org/) 

## API 

point(x, y)
circle(x, y, r)
rectangle(x, y, w, h)
ellipse(x, y, w, h)
line(x1, y1, x2, y2)
